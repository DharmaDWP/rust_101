## Create application

`cargo init .`

## Run application

`cargo run`

## Run a file

### With compile detail

`cargo run --bin a1`

## Without compile detail

`cargo run -q --bin a1`

## Type

### enum

### struct
used for telling object detail.

### impl
used for implement some function to struct.

### vec
same like array

### match
same like switch, but more strict
